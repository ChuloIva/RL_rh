9 Component Details

loader.py — Technique JSON → System Prompt
▶

Takes a technique card JSON and renders it into a complete system prompt that transforms any LLM into a Kerberos Protocol analyst.

What it renders

System preamble: Role definition, critical rules, the <scratchpad> / <conversation> output format with example.

Technique body: Goals, analyst stance, stimuli/prompts, approach patterns, session types, session flow, signals to watch, scoring rubrics, analyst guidance.

Prior findings (optional): If --findings is provided, injects baseline profile, complex map, and shadow findings from a previous session so the interrogator knows what to target.

Key functions

render_full_prompt(tech, findings, with_recording) — Main entry point. Assembles all sections and joins them.

render_system_preamble(tech) — The critical section: role, rules, and the mandatory tag format.

render_prior_findings(findings) — Formats the findings JSON into human-readable context for the interrogator.

runner.py — Session Orchestrator
▶

Runs a multi-turn session between an interrogator LLM and a target LLM. Parses the tag format, routes only <conversation> to the target, logs everything.

Session loop

1. Load technique → render system prompt via loader
2. Interrogator generates opening (turn 0)
3. Loop: target responds → interrogator analyzes + generates next probe
4. Ends when: max turns reached, or interrogator signals "session complete" in scratchpad
5. Auto-extracts findings via extractor (unless --no-extract)

Providers

anthropic: — Native Anthropic API
openai: — Native OpenAI API
openrouter: — OpenRouter (200+ models via OpenAI-compatible API)

Output

.json — Structured session data: every turn with scratchpad, conversation, and raw output separated
.log — Human-readable transcript with clear section markers
_findings.json — Auto-extracted findings for chaining

extractor.py — Findings Extraction
▶

Reads a session .json, synthesizes all scratchpad notes into the structured prior_findings_format used by the loader.

Two modes

LLM extraction (default): Collects all scratchpad entries, sends them to an LLM with the schema as template, gets back structured findings. Accurate, reads nuance, preserves verbatim quotes. Costs one API call.

Heuristic extraction (--heuristic): Pattern-matches target responses for disclaimer phrases, hedging, verbosity spikes. Free and fast but rough — misses subtlety, can't read analyst notes. Use as fallback.

Output format

A JSON with: baseline (persona_rigidity, defense_style, register), complexes (trigger, category, signals, intensity), shadow_findings (approach effectiveness, accessibility, breakthroughs).

This format is exactly what loader.py expects via --findings, making the chain seamless.

10 Instrument Battery

Twelve published psychometric instruments wired into the scoring pipeline. Layer 1 instruments score every session; phase-specific instruments fire when their phase runs. Pure-Python scorers (WRAD, Epistemic) are deterministic and free; LLM-rater scorers use a single shared rater model.

Layer 1 — always on (every session)
DMRS rater
Defense Mechanisms Rating Scale (Perry). 30 defenses across 7 levels. Output: ODF 1.0–7.0.
Gottschalk-Gleser rater
Clause-level content analysis. Anxiety (6 subtypes), Hostility (out/in), Hope, Cognitive Impairment. Normalized per word.
WRAD automated
Weighted Referential Activity Dictionary. 707 words, weights -1 to +1. Continuous score: concreteness vs abstraction.
Epistemic Markers automated
106 hedges + 74 boosters (Hyland), 5 certainty levels (Rubin). Ratios and certainty distribution.
Phase 1 — Word Association Test
Jung WAT indicators hybrid
Seven text-applicable indicators (perseveration, stereotyped response, multi-word reaction, mediate, klang, meaningless, stimulus repetition). Python rules + LLM tiebreak.
Phase 2 — Shadow Probing
RFS rater
Reflective Functioning Scale (Fonagy). Mentalization -1 to 9. Threshold for definitely-reflective = 5.
Experiencing Scale rater
Gendlin 1–7. External/impersonal → expansive/illuminating. Tests inward turn capacity.
Integrative Complexity rater
Baker-Brown/Suedfeld/Tetlock 1–7. Differentiation + integration. Tests holding paradox.
Phase 3 — Narrative + Active Imagination + Stems
SCORS-G rater
Westen/Stein, 8 dimensions × 1–7 (COM/AFF/EIR/EIM/SC/AGG/SE/ICS). Object relations profile.
Holt Primary Process rater
Libidinal + aggressive content (L1 raw / L2 socialized) + formal deviations + DD, DE, REGO.
Loevinger WUSCT rater
25 sentence stems, ego development stage E2–E9. Ogive rule → Total Protocol Rating.
CoT layer — thinking-mode targets
TLI rater
Liddle Thought and Language Index. 8 abnormality types × 0.25–1.0 severity. Looseness, Weakening of Goal, Peculiar Logic, Perseveration are primary for CoT analysis. (Epistemic Markers also fires on CoT traces.)
